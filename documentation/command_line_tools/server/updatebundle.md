# NAME

updatebundle - updates a bundle

# SYNOPSIS

**updatebundle** [OPTIONS] location


# DESCRIPTION

This command line tool updates a specific bundle.

# OPTIONS

**-h** *jmxHost*
: The JMX host

**-p** *jmxPort*
: The port

**-l** *jmxLogin*
: The optional login
 
**-pw** *jmxPassword*
: The optional password

# EXAMPLES

**updatebundle open-xchange-server**
