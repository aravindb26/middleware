# NAME

installbundle - installs a bundle

# SYNOPSIS

**installbundle** [OPTIONS] location


# DESCRIPTION

This command line tool install a specific bundle.

# OPTIONS

**-h** *jmxHost*
: The JMX host

**-p** *jmxPort*
: The port

**-l** *jmxLogin*
: The optional login
 
**-pw** *jmxPassword*
: The optional password

# EXAMPLES

**installbundle open-xchange-server**
