# NAME

refreshbundles - refresh a bundles

# SYNOPSIS

**refreshbundles** [OPTIONS]


# DESCRIPTION

This command line tool refreshes all bundles.

# OPTIONS

**-h** *jmxHost*
: The JMX host

**-p** *jmxPort*
: The port

**-l** *jmxLogin*
: The optional login
 
**-pw** *jmxPassword*
: The optional password
